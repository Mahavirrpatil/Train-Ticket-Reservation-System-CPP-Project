# Train Ticket Reservation System

This project is a simple Train Ticket Reservation System developed using C++.
The program allows employees to manage train records and passengers to reserve, view, and cancel train tickets through a menu-driven console interface.

## Features

* Add new trains
* Remove trains
* View available trains
* Reserve train tickets
* View reservation details
* Cancel reservations using password
* Automatic fare calculation based on distance between stations
* Menu-driven console application

## Stations Included

* CSMT
* Kalyan
* Nashik
* Manmad
* Pachora
* Jalgaon
* Bhusawal
* Raver
* Burhanpur
* Nepanagar
